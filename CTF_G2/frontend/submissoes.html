Two Sum

Dada uma lista de inteiros nums e um inteiro target, retorne os indices dos dois numeros cuja soma seja igual a target.

Voce pode assumir que cada entrada tera exatamente uma solucao, e voce nao pode usar o mesmo elemento duas vezes.

Voce pode retornar a resposta em qualquer ordem.

Exemplo 1:

Entrada: nums = [2, 7, 11, 15], target = 9

Saida: [0, 1]

Explicacao: nums[0] + nums[1] = 2 + 7 = 9, entao retornamos [0, 1].

Exemplo 2:

Entrada: nums = [3, 2, 4], target = 6

Saida: [1, 2]

Exemplo 3:

Entrada: nums = [3, 3], target = 6

Saida: [0, 1]

Restricoes:

  • 2 ≤ nums.length ≤ 104
  • -109 ≤ nums[i] ≤ 109
  • -109 ≤ target ≤ 109
  • Apenas uma resposta valida existe.

Assinatura da funcao:

def two_sum(nums: list[int], target: int) -> list[int]:
